AI Chat Bot App


This ReactJS-based chatbot application, powered by the ChatGPT API, provides a seamless and intuitive conversational experience. Designed with accessibility in mind, it features a clean, minimalistic, and neurodivergent-friendly UI, ensuring a distraction-free and user-friendly interface.

News App


A responsive news application using React.js and Vite, integrating Axios to fetch real-time articles from the GNews API across multiple categories with a clean, consistent interface, and engineered interactive navigation with a modal-based detailed view to enable busy users to quickly preview headlines.

Event Management App


A web application developed using Express.js and MongoDB enables users to create, view, and book events seamlessly while ensuring efficient data handling. It has a intuitive interface that supports event booking and an authentication system for a smooth and reliable experience.

Blogging App


This SaaS application is an AI-powered blogging platform that allows users to generate and manage blog content seamlessly. Built using Next.js for the frontend and backend, MongoDB for efficient data storage, and Tailwind CSS for a sleek and responsive UI, the app provides a modern and scalable solution for content creators.

Travel Planner


This Figma-designed prototype is built to provide a low-distraction, intuitive travel planning experience tailored for individuals with ADHD, autism, or anxiety. With a minimalist UI and customizable itineraries, it helps reduce travel-related stress while emphasizing inclusive and accessible design principles.

Smart Resume Builder


This AI-Based Resume Reviewer is a Figma-designed prototype that aids in building resume as well as provides instant AI-driven feedback including ATS score to help job applicants optimize their resumes. It features resume scoring, and industry-specific formatting recommendations, enhancing user experiences.
